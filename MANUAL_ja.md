# ユーザーマニュアル

このスケッチを書き込んだ HAGIWO MOD1 を初めて使う人向けの簡易マニュアルです。

## できること

このスケッチは、ゲート信号が HIGH の間、または本体ボタンを押し続けている間、D11 / F4 から矩形波のサイレン音を出力します。A0 で基本ピッチ、A1 で LFO の深さ、A2 で LFO の速さを調整します。LFO はピッチを変調し、D3 の LED は同じ LFO の動きを LFO 深さに応じた明るさで表示します。

## 接続

| MOD1 signal | Arduino pin | Role |
|---|---:|---|
| POT1 | A0 | VCOピッチ |
| POT2 | A1 | LFO 深さ |
| POT3 | A2 | LFO 速度 |
| F1 | A3 / D17 | ゲート入力 |
| F2 | D9 | キル入力 |
| F3 | D10 | LFO ポーズ入力 |
| F4 | D11 | オーディオ出力 |
| Button | D4 | 波形選択 / 手動ゲート |
| LED | D3 | LFO インジケータ |

F4 / D11 をミキサーや後段モジュールへ接続します。出力はラインレベル音声ではなく、Arduino からの 5V 矩形波です。外部オーディオ機器へ接続する場合は、ミキサー入力やアッテネーターを使い、音量を小さめから確認してください。ヘッドホンやパッシブスピーカーへ直接接続しないでください。

F1、F2、F3 に接続する外部信号は、0V から 5V の範囲内で、正の電圧のみにしてください。ユーロラックなどの負電圧を含む信号や、5V を超える信号を直接接続すると、回路が破損する可能性があります。必ず外部信号を適切な範囲に調整し、外部機器と接続するときは GND を共通にしてください。

## 基本操作

- POT1 / A0: 基本ピッチを変更します。
- POT2 / A1: LFO がピッチを変調する深さを変更します。LED の明るさ変化の深さにも反映されます。
- POT3 / A2: LFO 速度を変更します。左端付近でも LFO は停止せず、非常にゆっくり動き続けます。
- F1 / A3 / D17: HIGH の間だけ発音します。
- F2 / D9: HIGH の間、出力を強制ミュートします。ゲートやボタンより優先されます。
- F3 / D10: HIGH の間、LFO の動きを一時停止します。
- Button / D4: 0.23 秒未満の短押しで LFO 波形を変更します。0.23 秒以上押し続けると、押している間だけ手動で発音します。
- LED / D3: LFO の速さと深さを表示します。

LFO 波形は、SINE、SQUARE、SAWTOOTH、REVERSE SAWTOOTH の順に切り替わります。

## 初回テスト

1. POT1、POT2、POT3 を中央付近にします。
2. F4 / D11 をミキサーやアンプへ接続し、音量を小さめにします。
3. ボタンを 0.23 秒以上押し続けて、押している間だけ音が出ることを確認します。
4. POT1 を回して、ピッチが変わることを確認します。
5. POT2 と POT3 を回して、ピッチの揺れと LED の動きが変わることを確認します。
6. ボタンを短く押して、LFO 波形が変わることを確認します。
7. F1 / A3 に 5V ゲートを入れて、ゲートが HIGH の間だけ音が出ることを確認します。
8. F2 / D9 に HIGH を入れてミュート動作を確認し、F3 / D10 に HIGH を入れて LFO ポーズ動作を確認します。

## トラブルシューティング

- F4 / D11 から出力を取っているか確認してください。
- F1 / A3 のゲートが LOW の場合、ボタンを 0.23 秒以上押して、手動発音できるか確認してください。
- F2 / D9 が HIGH の場合、出力は強制ミュートされます。
- ミキサーやアンプの入力音量が下がっていないか確認してください。
- MOD1 と Arduino Nano に正しく 5V と GND が供給されているか確認してください。
