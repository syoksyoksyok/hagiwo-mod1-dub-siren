# HAGIWO MOD1 Dub Siren
HAGIWO MOD1 ベースのダブサイレン用 Arduino Nano / ATmega328P スケッチです。
これは HAGIWO MOD1 ハードウェア向けのファームウェアです。
HAGIWO MOD1 は HAGIWO さんによって設計されています。

現在のバージョン: `v1.0.0`

## 必要なもの
- HAGIWO MOD1 ハードウェア
- Arduino Nano / ATmega328P
- 通常利用には Arduino IDE、ビルド確認には PlatformIO

## 書き込み
1. このリポジトリをダウンロード、または clone します。
2. Arduino IDE で `hagiwo-mod1-dub-siren.ino` を開きます。
3. `Arduino Nano` と `ATmega328P` を選択します。
4. シリアルポートを選択します。
5. `Upload` をクリックします。
6. 書き込みに失敗する場合は、`ATmega328P (Old Bootloader)` を試してください。

詳しい書き込み手順とトラブルシューティングは `UPLOAD_STEPS_ja.md` を参照してください。

## できること
このスケッチは HAGIWO MOD1 をダブサイレンマシンにします。

本体ボタンを押し続けている間、または F1ジャック(A3 / D17) に HIGH が入力されている間、F4ジャック(D11) から矩形波のサイレン音を出力します。
POT1で VCO の基本ピッチ、POT2で LFO の深さ、POT3(A2)で LFO の速さを調整します。
LFO はピッチを変調し、LED は LFO の動きを LFO 深さに応じた点滅で表示します。

## 接続
| パネル表記 | Arduino ピン | 役割 |
|---|---:|---|
| POT1 | A0 | VCO ピッチ |
| POT2 | A1 | LFO 深さ |
| POT3 | A2 | LFO 速度 |
| F1ジャック | A3 / D17 | ゲート入力(発音) |
| F2ジャック | D9 | キル入力 |
| F3ジャック | D10 | LFO ポーズ入力 |
| F4ジャック | D11 | オーディオ出力 |
| Button | D4 | 波形選択 / 手動ゲート |
| LED | D3 | LFO インジケータ |

F4ジャック(D11)をミキサーや後段モジュールへ接続します。
出力はラインレベル音声ではなく、Arduino からの 5V 矩形波です。
外部オーディオ機器へ接続する場合は、ミキサー入力やアッテネーターを使い、音量を小さめから確認してください。
ヘッドホンやパッシブスピーカーへ直接接続しないでください。

F1、F2、F3 に接続する外部信号は、0V から 5V の範囲内で、正の電圧のみにしてください。
ユーロラックなどの負電圧を含む信号や、5V を超える信号を直接接続すると、回路が破損する可能性があります。
必ず外部信号を適切な範囲に調整し、外部機器と接続するときは GND を共通にしてください。

## 基本操作
- POT1(A0): 基本ピッチを変更します。
- POT2(A1): LFO がピッチを変調する深さを変更します。LED の明るさ変化の深さにも反映されます。
- POT3(A2): LFO 速度を変更します。左端付近でも LFO は停止せず、非常にゆっくり動き続けます。
- F1ジャック(A3 / D17): HIGH の間だけ発音します。
- F2ジャック(D9): HIGH の間、出力を強制ミュートします。ゲートやボタンより優先されます。
- F3ジャック(D10): HIGH の間、LFO の動きを一時停止します。
- Button(D4): 0.23 秒未満の短押しで LFO 波形を変更します。0.23 秒以上押し続けると、押している間だけ手動で発音します。
- LED(D3): LFO の速さと深さを表示します。

LFO 波形は、SINE、SQUARE、SAWTOOTH、REVERSE SAWTOOTH の順に切り替わります。


## トラブルシューティング
- F4ジャック(D11)から出力を取っているか確認してください。
- F1ジャック(A3 / D17)のゲートが LOW の場合、ボタンを 0.23 秒以上押して、手動発音できるか確認してください。
- F2ジャック(D9)が HIGH の場合、出力は強制ミュートされます。
- ミキサーやアンプの入力音量が下がっていないか確認してください。
- MOD1 と Arduino Nano に正しく 5V と GND が供給されているか確認してください。
- 書き込み時のトラブルは `UPLOAD_STEPS_ja.md` を参照してください。

## 注意
MOD1 ハードウェア上では、F2ジャック(D9)は A4 と信号経路を共有し、F3ジャック(D10)は A5 と信号経路を共有します。
このスケッチでは D9 と D10 を高インピーダンスのデジタル入力としてのみ使用します。
